# 🏥 Multi-Modal AI Healthcare Management System

## 📝 Overview
This project is an advanced, touchless AI system designed to automate and secure the patient scheduling process. Instead of a traditional interface, the system integrates computer vision and natural language processing to create a fully hands-free experience. It begins with secure Face Authentication, after which users can seamlessly interact with the system using a conversational Voice Assistant, or switch to dedicated Voice Control and Facial Control modes for complete system navigation.

## ✨ Key Features
* **Biometric Security (Face Authentication):** Secure, real-time login and user verification using facial recognition technology.
* **Smart Voice Assistant:** An NLP-driven conversational interface specifically tailored to understand patient intent and automate appointment scheduling logic.
* **Multi-Modal Control Systems:** 
  * **Voice Control:** Allows users to navigate the system and execute commands entirely via speech.
  * **Facial Control:** A vision-based interface that tracks facial movements/gestures to control the system without physical contact.
* **Automated Healthcare Scheduling:** Intelligently manages time slots, checks availability, and registers bookings.

## 🛠️ Tech Stack
* **AI & NLP:** [MARBERT, BILSTM]
* **Computer Vision:** [FaceRecognition, OpenCV, MediaPipe, dlib]
* **Speech Processing:** [SpeechRecognition, PyAudio]
* **Backend/Database:** [Multi Threading, Excel]
